Motions of Modular Volumetric Actuators

6.832 - Underactuated Robotics

In the spring of 2020, I developed a simulation for robotic systems I was working on in Drake. It was the final project for a class at MIT, 6.832 Underactuated Robotics. Using the C++ API for Drake, I simulated the robots and their modular configurations.In the image, for instance, you can see eight spherical actuators in a cube formation, which expand and contract syncronously to roll over.

You can find the code for the simulation here: Github

It was a challenging project, since modeling compliance for these systems is complex in Drake. Still, I was able to create a framework to generate and control these actuators with the help of some recently added features such as LinearBushingRollPitchYaw force elements.

Below is the abstract for my final report, of which the full text is available here: Full text

In robotics, actuation via volume change has become popular and critical to many systems. A spherical electromechanical actuator was developed in 2019 by the Distributed Robotics Laboratory in CSAIL. These actuators are modular and can be linked to form structures and achieve certain dynamic behaviors. Yet, forming large structures physically is difficult and expensive. In this paper, I introduce a framework for generating structures from spherical volumetric actuators to simulate, and discuss how analysis of these systems fosters educated decisions about which structures to generate, and how to control them.
